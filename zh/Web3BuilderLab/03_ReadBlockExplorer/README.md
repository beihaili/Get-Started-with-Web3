# 如何阅读区块浏览器

> 🚧 本课程正在编写中，欢迎 [参与贡献](https://github.com/beihaili/Get-Started-with-Web3/blob/main/CONTRIBUTING.md)！

## 你将学到

- 掌握 Etherscan 等区块浏览器的核心功能与页面布局
- 解读交易详情：from、to、value、gas、input data
- 分析合约交互记录与事件日志（Event Logs）
- 利用区块浏览器排查链上问题和追踪资金流向

## 前置知识

- 完成「Web3 快速入门」模块
- 了解以太坊交易和智能合约的基本概念
- 有过发送链上交易的经验

## 课程大纲

1. 区块浏览器是什么？为什么它是 Web3 的必备工具
2. Etherscan 页面导航：区块、交易、地址、合约
3. 解读一笔转账交易的完整信息
4. 解读合约调用交易与 Input Data 解码
5. 分析 Event Logs 与内部交易（Internal Transactions）
6. 实战：追踪一笔 DeFi 交易的完整链路
