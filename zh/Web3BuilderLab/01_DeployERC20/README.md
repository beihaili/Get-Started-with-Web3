# 从零部署 ERC-20 代币

> 🚧 本课程正在编写中，欢迎 [参与贡献](https://github.com/beihaili/Get-Started-with-Web3/blob/main/CONTRIBUTING.md)！

## 你将学到

- 理解 ERC-20 代币标准的核心接口与设计理念
- 使用 Solidity 编写一个完整的 ERC-20 合约
- 通过 Foundry / Hardhat 将合约部署到测试网
- 在 MetaMask 中添加并管理自定义代币

## 前置知识

- 完成「Web3 快速入门」模块
- 了解以太坊账户与交易基础概念
- 安装好 MetaMask 钱包并连接测试网

## 课程大纲

1. ERC-20 标准简介与核心函数解析
2. 编写 ERC-20 合约：totalSupply、balanceOf、transfer、approve、transferFrom
3. 配置开发环境（Foundry / Hardhat）
4. 编译、测试与部署到 Sepolia 测试网
5. 在区块浏览器上验证合约
6. 在 MetaMask 中添加代币并完成一笔转账
