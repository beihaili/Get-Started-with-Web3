## 内容总览 / Content Overview

### Part 1: Web3 快速入门（6 讲）

从零开始，6 讲完成 Web3 核心操作。无需编程基础。

| 讲次 | 主题 | 你将学会 |
|------|------|----------|
| 01 | [创建你的第一个 Web3 身份](./zh/Web3QuickStart/01_FirstWeb3Identity/README.md) | 安装钱包、创建地址、备份助记词 |
| 02 | [进行你的第一笔 Web3 交易](./zh/Web3QuickStart/02_FirstWeb3Transaction/README.md) | 转账、Gas 费、交易确认 |
| 03 | [构建你的第一个 Web3 DApp](./zh/Web3QuickStart/03_FirstWeb3Dapp/README.md) | 连接钱包、与智能合约交互 |
| 04 | [有用的 Web3 站点](./zh/Web3QuickStart/04_UsefulWeb3Sites/README.md) | 区块浏览器、DeFi 工具、信息聚合 |
| 05 | [发行你的第一个代币](./zh/Web3QuickStart/05_LaunchYourFirstToken/README.md) | ERC-20 代币发行全流程 |
| 06 | [Web3 安全防护](./zh/Web3QuickStart/06_Web3Security/README.md) | 识别骗局、保护资产、安全最佳实践 |

### Part 2: 比特币技术深潜（21 讲）

按 **5 层技术架构** 组织，从密码学基础到生态应用的完整技术栈。

```
  💼 应用层 (第15-21讲)     钱包、RPC 开发、脚本系统、Ordinals、DeFi 跨链
  🤝 共识层 (第12-14讲)     工作量证明、难度调整、分叉与 BIP 流程
  🌐 网络层 (第09-11讲)     Bitcoin Core 节点、P2P 协议、网络安全
  📊 数据层 (第03-08讲)     交易结构、多签、SegWit、Taproot、高级交易
  🔐 密码学层 (第01-02讲)   哈希函数、数字签名、椭圆曲线、HD 钱包
```

> 完整目录和学习路径建议：[比特币技术架构总览](./zh/GetStartedWithBitcoin/README.md)

## 平台特色 / Platform Features

**[在线学习平台](https://beihaili.github.io/Get-Started-with-Web3/)** 不只是文档阅读器：

- **学习进度追踪** — 自动记录每课完成状态

- **经验值与等级** — 每完成一课获得 XP，见证自己的成长
- **AI 辅导** — 基于当前课程内容的智能问答

## 快速开始 / Quick Start

**在线学习（推荐）：** 访问 https://beihaili.github.io/Get-Started-with-Web3/

**本地开发：**
```bash
git clone https://github.com/beihaili/Get-Started-with-Web3.git
cd Get-Started-with-Web3
npm install
npm run dev
```

**直接阅读 Markdown：** 从 [第 01 讲：创建你的第一个 Web3 身份](./zh/Web3QuickStart/01_FirstWeb3Identity/README.md) 开始
