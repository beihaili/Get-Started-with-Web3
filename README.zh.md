## 为什么做这个项目

Web3 的学习资料长期分散在钱包教程、协议文档、安全提醒、开发工具和碎片化文章里。这个项目把这些内容整理成一条可以持续学习、可以贡献、也可以被 AI Agent 检索和引用的开源课程路径。

如果这个项目帮你理解 Web3，请给 [GitHub 仓库点一个 star](https://github.com/beihaili/Get-Started-with-Web3)，这会让更多学习者看到它。

## 适合谁

| 角色             | 可以用它做什么                                                                       |
| ---------------- | ------------------------------------------------------------------------------------ |
| **完全新手**     | 创建钱包、完成第一笔交易、理解 Gas、识别常见骗局。                                   |
| **Web3 Builder** | 从概念进入 DApp、智能合约、区块浏览器、Bitcoin RPC、DeFi、L2 和 DAO 工具。           |
| **研究者**       | 系统复习 Bitcoin、Ethereum、DeFi、L2、DAO、跨链桥、稳定币和智能账户。                |
| **AI Agent**     | 通过 `llms.txt`、JSON artifacts 和本地 MCP server 搜索、读取、引用和组合课程上下文。 |
| **赞助商**       | 支持开源 Web3 教育，并在学习者选择默认工具前建立可信触达。                           |

## 项目边界

本仓库首先是课程内容平台和 AI-readable 知识层。后续会增加少量现代 Web3 实验，但这些实验用于教学，不是生产级 DApp 脚手架。

本项目不提供投资建议、代币推广、交易所导流或虚拟货币业务服务。钱包、SIWE、x402、链上证书、赞助和付费工具相关内容，必须在真实上线并验证前明确标注为 demo、metadata、draft 或 future plan。

## 快速开始

```bash
git clone https://github.com/beihaili/Get-Started-with-Web3.git
cd Get-Started-with-Web3
npm install
npm run dev
```

开发服务器会运行在 `http://localhost:5173/Get-Started-with-Web3/`。

常用命令：

```bash
npm run build      # 生产构建、OG 图片、sitemap、预渲染
npm test           # Vitest 测试
npm run lint       # ESLint 检查
npm run ai:verify  # 验证公开 AI 入口和 x402 元数据
```

## AI Agent 与 MCP

本仓库提供面向 AI Agent 的机器可读内容层：

```bash
npm run ai:index   # 重新生成 ai/manifest.json、ai/content-index.json、ai/llms.txt
npm run ai:publish # 复制 AI artifacts 到 public/
npm run ai:verify  # 检查公开 AI 入口和 x402 元数据
npm run mcp:web3   # 启动本地只读 stdio MCP server
```

Agent 使用方式：

```bash
npm run mcp:web3
```

如果你的 MCP client 支持 `mcpServers` JSON 配置，可以复制下面的配置，并把 `cwd` 替换成本仓库的本地绝对路径：

```json
{
  "mcpServers": {
    "get-started-with-web3": {
      "command": "npm",
      "args": ["run", "mcp:web3"],
      "cwd": "/absolute/path/to/Get-Started-with-Web3"
    }
  }
}
```

然后在 MCP client 中调用：

- `search_web3_content`: 搜索课程和术语。
- `read_web3_lesson`: 读取带引用的课程内容。
- `get_learning_path`: 获取新手、Builder、研究者或投资者学习路径。
- `lookup_web3_glossary`: 查询 Web3 术语。
- `compose_web3_context`: 组合带引用的上下文包。
- `list_monetizable_tools`: 查看未来 x402-ready 付费工具元数据。

本地 MCP 工具保持只读，不执行支付验证、链上签名或链上操作。x402 字段只是为未来远程付费工具预留的元数据。

## 技术栈

- **Frontend:** React 19 + Vite + Tailwind CSS
- **State:** Zustand
- **Routing:** React Router
- **Content:** `zh/` 与 `en/` 下的 Markdown
- **Rendering:** react-markdown + remark-gfm + rehype-raw
- **AI Tutor:** Gemini，使用用户自带 API key
- **Agent layer:** `@modelcontextprotocol/sdk`、生成的 JSON artifacts、`llms.txt`
- **SEO:** sitemap、OG 图片生成、预渲染
- **Testing:** Vitest + happy-dom + Testing Library
- **CI/CD:** GitHub Actions 到 GitHub Pages
