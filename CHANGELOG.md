# Changelog

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 格式。

## [1.0.0] - 2026-02-09

### Added
- React 19 驱动的交互式学习平台
- 5 个课程模块：Web3 入门、BTC 密码学/数据层、BTC 网络/共识层、BTC 应用层、Web3 深度思考
- 30 节课程内容（6 节 Web3 + 21 节比特币 + 3 节思考）
- AI 辅导助手（Gemini API）
- 模糊搜索功能（Fuse.js）
- 学习进度追踪与经验值系统
- 徽章成就系统
- 学习连续天数统计
- Quiz 测验系统
- 自定义 Markdown 渲染器（支持代码高亮、表格、图片）
- 响应式设计（桌面端 + 移动端）
- PWA 支持（Service Worker 离线缓存）
- 无障碍支持（ARIA 标签、键盘导航）
- GitHub Pages 自动部署
- Vitest 单元测试框架
- 社区贡献指南与模板
