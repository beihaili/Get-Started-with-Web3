# 构建你的第一个 DApp

> 🚧 本课程正在编写中，欢迎 [参与贡献](https://github.com/beihaili/Get-Started-with-Web3/blob/main/CONTRIBUTING.md)！

## 你将学到

- 理解 DApp 的前端与智能合约交互架构
- 使用 ethers.js/viem 连接钱包并读写链上数据
- 从零搭建一个具有完整交互流程的 DApp 前端
- 处理交易状态、错误提示与用户体验优化

## 前置知识

- 完成「从零部署 ERC-20 代币」课程
- 具备基本的 React 或前端开发经验
- 了解 JSON-RPC 和 Provider 的基本概念

## 课程大纲

1. DApp 架构概览：前端 + 钱包 + 合约
2. 使用 ethers.js/viem 连接 MetaMask
3. 读取链上数据：余额查询与事件监听
4. 发起交易：调用合约写入函数
5. 交易状态管理与用户提示
6. 完整项目实战：构建一个代币转账 DApp
