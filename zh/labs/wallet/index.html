钱包互操作实验室

一个聚焦钱包发现、网络上下文和消息签名的实验入口。第一阶段保持静态，等钱包技术栈评审后再接入依赖。

当前状态

脚手架已就绪。当前页面尚未安装钱包依赖，因此不会执行钱包连接、签名、交易或认证流程。

MVP 范围

  • 连接和断开钱包
  • 展示账户、链名称和 chain ID
  • 在批准的网络集合中切换
  • 签名一条教学消息，并在本地查看结果

隐私边界

实验室不得把原始钱包地址、签名、已签消息、provider 对象或交易 payload 发送到 analytics。

下一步实现切片

先补路由测试，并保持页面无依赖。wagmi、viem、RainbowKit 或 Reown AppKit 只在后续实现 PR 中安装。
